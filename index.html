General API json-rpc 2.0 Information

  • The base endpoint is: wss://???:???

  • Streams can be accessed either in a single raw stream or in a combined stream

  • Raw streams are accessed at /ws/<streamName>

  • Combined streams are accessed at /stream?streams=<streamName1>/<streamName2>/<streamName3>

  • Combined stream events are wrapped as follows: {"stream":"<streamName>","data":<rawPayload>}

  • All symbols for streams are lowercase

  • A single connection to stream.matrixbot.io is only valid for 24 hours; expect to be disconnected at the 24 hour mark

  • The websocket server will send a ping frame every 3 minutes. If the websocket server does not receive a pong frame back from the connection within a 10 minute period, the connection will be disconnected. Unsolicited pong frames are allowed.

  • Sample request:

    {"method": "methodName",
"id": "number",
"command": "command"
"params": [arg1, arg2, ...]}

  • All endpoints return either a JSON object or array.

  • All time and timestamp related fields are in milliseconds.

Error Codes
  • Any endpoint can return an ERROR

Sample Payload below:

{"result": null,
 "error": {"name": "JSONRPCError",
           "code": (number 100-999),
           "message": "Some Error Occurred",
           "error": "whatever you want\n(a traceback?)"},
 "id": "number"}

200 confirmation email sent, refferal accepted
201 confirmation email sent, refferal not accepted
-100 email exist
-102 wrong email
-103 wrong passw

  "userid": integer,     // user id
}

Bot

create

params:

{
  "name": string,   // unique bot name
  "market": string // market
  "symbol": string // trade pair
  "api": string // api key
  "strategy": string // type strategy
  "limit": float // trade amount for bot
  "cover": float // grid cover
  "profit": float //profit
  "invest": float // % icrease limit after profit
  "matrix": // matrix template name or array
      {
        "template":"name" // or float array[0..40]
        "arc": float // grid arc factor
      }
  "start":
      {
        "template":"name" // start filters template name or array of filters[0..20]
      }
  "xpair": // empty (no change pair) or params
      {
        "list": // empty without list, 0 - whitelist, 1 - blacklist
            {
              "[0..30]" // array of symbols
            },

      },
  "managment": // name of template or array of rules
      {
        "template":"name" // or float array[0..40]
      }
}

codes:

code success error data
200 bot created bot id
-100 name taken
-102 wrong data description

getinfo

params:

{
  "botid" : integer // bot id
}

codes:

code success error data
200 bot info full info
-100 wrong id

full answer

{
  "status": string, // active, stopped, waiting for run, waiting for stop,
                   //  on start filters, searching pair
  "botid": integer, // bot id
  "created": timestamp, // bot created
  "modificate": timestamp, // bot modificated
  "started": timestamp, // last bot started
  "rounds": [x, y], // rounds counter: X complected, Y canceled
  "deep": integer, // for active bot: X orders filled from total
  "total": float, // total profit in usd
  "filter": string, // last used filter
  "name": string,   // unique bot name
    "market": string,   //market name
    "api": string, // api name
  "symbol": string, // current trade pair
  "strategy": string, // type strategy
  "limit": float, // trade amount for bot
  "leverage": integer, // leverage
  "cover": float, // grid cover
  "profit": float, //profit
  "reinvest": float,// % icrease limit after profit (default 0.1)
  "matrix": // matrix template name or array
      {
        "template":"name", //
        "grid": [array of floats] // grid [0.1, 0.5... 3,4]
        "arc": float, // grid arc factor
        "orders": integer, // orders
        "placeby": integer, // partial placing of orders
        "factor": float // matrix factor
      },
  "start":
      {
        "template":"name", // start filters template name or array of filters[0..20]
        "offset": float, // offset % start order (default 0.25)
        "reset": float, // reset % grid if price gone (default 0.5)
        "sleep": float, // sleep minutes after round finished (default 0.5)
      },
  "xpair": // change pairs
      {
        "enable": boolean, // 0 - disable, 1 - enable
        "template": "name" // template
        "type": boolean, // 0 - whitelist, 1 - blacklist, empty - nolist
        "list": [array of strings] // array of symbols (max 50)
      },
  "managment": // name of template or array of rules
      {
        "template":"name",
        "rules": [] // array of rules (max 20)
      },
}

{

  "end": timestamp // end time
}

codes:

code success error data
200 rounds info array of objects

full answer

{
  "rounds":
    {
    "id": integer // id of round
    "start": timestamp // start round time
    "end": timestamp // end round time
    "status": string // active, comlected or canceled
    "symbol": string // pair
    "orders": [x, y] // X filled from Y
    "limit": float // money amount in round (usd)
    "profit": float // profit (usd)
    "filter": string // last used filter
    }
}
